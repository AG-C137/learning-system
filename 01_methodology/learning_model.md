# Learning Model

## 1. Принцип

Изучение через систему, а не через хаотичное потребление.

Среда → Процесс → Проект → Рефлексия → Корректировка.

---

## 2. Рабочий цикл

1. Постановка задачи
2. Минимальная реализация
3. Осознанный commit
4. Анализ результата
5. Фиксация выводов

---

## 3. Правила

- Один шаг за раз
- Никакой магии без понимания
- Всё важное фиксируется в репозитории
- Эксперименты отделены от проектов

---

## 4. Цель

Построение собственной операционной системы мышления.
