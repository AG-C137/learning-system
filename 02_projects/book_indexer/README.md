# book_indexer

Локальный RAG по книгам.

## Команды

index:
book-index index books/

search:
book-index search "запрос"

ask:
book-index ask2 "вопрос"

## Возможности

* поиск по книгам
* фильтр по автору / книге
* генерация ответа через LLM

## Стек

* Ollama
* SQLite
* Python
